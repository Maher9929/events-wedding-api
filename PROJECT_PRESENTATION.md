# DOUSHA - Marketplace de Services de Mariage (Demo)

Ceci est la présentation technique du projet **DOUSHA**, une plateforme complète facilitant l'organisation de mariages.

## 🚀 Fonctionnalités Clés
- **Système Multi-Rôles :** Client, Prestataire (Provider), et Administrateur.
- **Gestion d'Événements :** Création d'événements, gestion de listes d'invités et suivi de budget.
- **Messagerie Temps Réel :** Chat direct entre clients et prestataires intégré.
- **Système de Réservation :** Workflow complet de la sélection du service au paiement sécurisé.
- **Paiements Intégrés :** Support de Stripe pour les acomptes et règlements complets.
- **Tableaux de Bord :** Statistiques dynamiques pour les prestataires et l'administration.
- **Localisation :** Support multi-langues (Arabe, Français, Anglais).

## 🛠️ Architecture Technique
- **Frontend :** React.js + Tailwind CSS + i18next
- **Backend :** NestJS (TypeScript)
- **Base de Données :** Supabase (PostgreSQL) + Auth
- **Paiement :** API Stripe
- **Monitoring :** Sentry intégré
